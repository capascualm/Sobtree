# Sobtree Holding - Bitcoin-first Treasury

**https://sobtree.com**

Este es el repositorio de la página web oficial de **Sobtree Holding SL**, una estructura de tesorería privada con un enfoque **Bitcoin-first**, orientada a la construcción de resiliencia financiera y preservación de patrimonio multigeneracional.

## Estructura del site

- `index.html` → página principal en español  
- `index_en.html` → versión en inglés  
- `robots.txt` → configuración para indexación en buscadores  
- `sitemap.xml` → mapa del sitio para SEO (declarado en Google Search Console)  
- `favicon.ico` → icono de la página  
- `sobtree_transparent.png` → logotipo de la marca  

## Objetivo

El objetivo de esta página es explicar de forma simple y clara la filosofía de Sobtree Holding:

- Construcción de una **tesorería antifrágil**  
- Uso de **hard assets** como pilar de resiliencia  
- Aplicación de principios **Bitcoin-first** en la gestión patrimonial  

Además, se utiliza como plataforma para atraer suscriptores a la newsletter de referencia del proyecto.

## SEO y optimización

El site está optimizado para SEO básico:

- `robots.txt` permite el indexado completo del site  
- `sitemap.xml` facilita la indexación de las páginas principales  
- Etiquetas meta para SEO y Open Graph implementadas  
- Velocidad de carga alta (web estática servida desde Netlify)

## Deploy

El site se despliega automáticamente desde GitHub a **Netlify**.  
Cada commit a `main` actualiza la página en producción: [https://sobtree.com](https://sobtree.com).

## Notas

- Verificación de dominio realizada en Google Search Console.  
- Mantener la meta tag de verificación en el `index.html` para asegurar el tracking SEO.  
- Si se añaden nuevas páginas en el futuro, deben incluirse en el `sitemap.xml`.
